Universal Designs for Olympics

Unique and cleare icons for Winter Olympics

In this assignment, I created four pictograms to represent different events in the Winter Olympic Games. Based on research, I learned how to represent each game clearly and create an effective graphic style.

Winter Olympic Games

Through this challenge, we learned that simplicity is the most important in creating universal icons and designs that are recognizable to everyone. We examined the information about all of the Olympic games and, we chose summer or winter games to create unique and new designs. The final logo will serve as a unique mark for the Winter Olympics.

About

This project was created as part of a school project. The course was ART 127 Graphic Symbolism and the assignment was to create universal icons for Olympic Games.
